# Title (`seo-title`)

## Syntax

```html
<seo-title
  value="myTitle"
></seo-title>
```

## Props

- __value__ : The title of the page

## Impact

- __Google__: Sets the title of the search result
- __Browser__: Sets the title of the window/tab
- __OpenGraph/Facebook search__: Sets the Title of your shared item
